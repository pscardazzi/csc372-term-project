Reflection write-up
Design Choices: React was an interesting frontend and I wanted more experience with it, Express was the one backend I was more familiar with, postgreSQL I'm also got experience with. I wanted to use ZenQuotes API so that a person could feel calm or inspired while writing, and Render was the deployment site I knew from the class.
Challenges: Definetely quote API integration and deploying the site. You'd think it's simple and easy, and then you get hit with error after error. It got really annoying sometimes.
Learning Outcomes: It's tougher than I thought! Didn't think i would be loaded with so many issues, but I sure did. I really need to improve on time management when it comes to projects like these. But nonetheless, I do feel a bit wiser when it comes to HTML programming. I think I can build a decent personal site now.
Future Work: A quotes.json so that I can emulate the API I wanted to use, and hopefully proper deployment.
