Reflection Write-up<br><br>
Design Choices: React was an interesting frontend and I wanted more experience with it, Express was the one backend I was more familiar with, postgreSQL I also got experience with. I wanted to use ZenQuotes API so that a person could feel calm or inspired while writing, and Render was the deployment site I knew from the class.<br><br>
Challenges: Definetely quote API integration and deploying the site. You'd think it's simple and easy, and then you get hit with error after error. It got really annoying sometimes.<br><br>
Learning Outcomes: Full-stack development is tougher than I thought! Didn't think I would be hit with so many issues, but I sure was. I really need to improve on time management when it comes to projects like these. Regardless, I do feel a bit wiser when it comes to web development. I think I can build a decent personal site now!<br><br>
Future Work: A quotes.json so that I can emulate the API I wanted to use, posts being "locked" to the user who posted them, and hopefully proper deployment.<br><br>
